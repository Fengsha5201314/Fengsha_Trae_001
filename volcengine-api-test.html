🔧 API配置

📋 获取API密钥步骤：

  1. 访问 火山引擎控制台
  2. 注册并登录账号
  3. 在密钥管理中创建AccessKey和SecretKey
  4. 将AccessKey和SecretKey填入下方表单

💡 CORS代理说明

由于浏览器安全策略，直接调用火山引擎翻译API可能被CORS阻止。选择一个代理服务可以解决此问题：
• cors-anywhere: 免费但有使用限制
• allorigins: 稳定的免费代理服务
• 自定义代理: 使用您自己的代理服务器
