  4. 将APP ID和密钥填入下方表单

💡 CORS代理说明

由于浏览器安全策略，直接调用百度翻译API可能被CORS阻止。选择一个代理服务可以解决此问题：
• cors-anywhere: 免费但需要先解锁访问权限 👆 点击解锁
• allorigins: 稳定的免费代理服务（推荐）
• 自定义代理: 使用您自己的代理服务器
• 注意: 公共代理服务可能不稳定或有速率限制，建议用于测试目的。

⚠️ CORS代理访问被拒绝

检测到 cors-anywhere 服务返回403错误。请按以下步骤解决：
1. 点击这里访问解锁页面
2. 点击页面上的 "Request temporary access to the demo server" 按钮
3. 或者切换到 "allorigins" 代理服务（更稳定）
4. 重新尝试翻译

🔧 故障排除指南

❌ 错误码 52003 - 未授权用户

原因：APP ID或APP Key错误，或服务未开通

解决方案：

  • 检查APP ID和APP Key是否正确填写
  • 登录百度翻译控制台确认服务状态
  • 新用户需要先开通通用翻译API服务
  • 确认账户余额充足（新用户有免费额度）

🚫 CORS代理访问被拒绝

原因：cors-anywhere服务需要手动解锁

解决方案：

  • 访问cors-anywhere解锁页面
  • 点击"Request temporary access"按钮
  • 或切换到allorigins代理（推荐）

🌐 网络连接问题

解决方案：

  • 检查网络连接是否正常
  • 尝试切换不同的CORS代理
  • 检查防火墙或代理设置
